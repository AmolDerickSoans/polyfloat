# PolyCLI

Agentic CLI/TUI terminal for prediction markets (Polymarket + Kalshi).

## Features
- CLI for market data and trading
- TUI Dashboard for real-time monitoring
- Autonomous trading agents via LangGraph (Powered by Google Gemini)

## Installation
```bash
pip install .
```

## Environment Variables
- `GOOGLE_API_KEY`: Required for agent features.
