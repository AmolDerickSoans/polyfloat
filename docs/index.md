# Welcome to PolyCLI

PolyCLI is an agentic terminal for prediction markets, supporting **Polymarket** and **Kalshi**.

## Key Features

-   **Unified Interface**: Trade on multiple platforms from a single CLI.
-   **TUI Dashboard**: Real-time monitoring of markets and positions.
-   **Autonomous Agents**: Deploy LangGraph-powered bots to trade for you.
-   **Arbitrage Scanner**: Detect price discrepancies instantly.

## Installation

```bash
pip install polycli
```

## Quick Start

Launch the dashboard:
```bash
poly dashboard
```
